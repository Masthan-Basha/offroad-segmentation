# offroad-segmentation
Offroad-Segmentation is a deep learning project for semantic segmentation of off-road terrain using DeepLabV3 with a ResNet-50 backbone. The model is trained to classify terrain regions from RGB images, enabling scene understanding for autonomous off-road navigation and robotics applications.
